
# OpenGent AI Platform

Modern AI startup landing page and demo platform.

Features
- React + TypeScript (Vite)
- Animated AI architecture visualization
- Agent simulation dashboard
- AI pipeline visualization
- Stripe waitlist integration placeholder
- Docker deployment
- Raspberry Pi compatible (Node container)
- Investor‑grade landing page layout

Run locally

npm install
npm run dev

Build

npm run build

Docker

docker build -t opengent .
docker run -p 3000:3000 opengent
